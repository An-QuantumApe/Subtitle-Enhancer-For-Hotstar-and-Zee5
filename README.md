# Subtitle Enhancer For Hotstar and Zee5

Customize subtitles on ZEE5 and JioHotstar.

## Features

- Subtitle size adjustment
- Subtitle position adjustment
- Line-height adjustment
- Bold subtitles
- Black outline toggle
- Automatic subtitle background removal
- Works in fullscreen
- Settings saved automatically

## Installation

1. Download or clone this repository.
2. Open `chrome://extensions`
3. Enable **Developer Mode**.
4. Click **Load unpacked**.
5. Select the extension folder.

## Supported Sites

- ZEE5
- JioHotstar / Hotstar

## Permissions

The extension only uses:

- `storage` – saves your settings locally in Chrome.

No analytics, tracking, network requests, or telemetry.
